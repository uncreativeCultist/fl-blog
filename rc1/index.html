Release Canidate #1 Released!

April 15, 2025 - FL Team


Hello there everyone! Writing a quick changelog for our first release canidate! There's not a lot of major additions, but it's important changes none the less!



Additions

  • Added missing TDM UI Elements.



Balance Changes

  • Made respawning in TDM faster.
  • Gordon can no longer eject himself into orbit using an SMG Grenade.



Bug Fixes

  • Fixed Gordon Civilian posing when crouching after a loss.
  • Fixed Some sounds not working for Gordon's weapons.
  • Fixed missing ammo packs on HL2DM maps.
  • Fixed Gordon's AR2 Upgrade not working properly.
  • Fixed Gordon's armor HUD being stretched.
  • Fixed Fire Timer Multiplication not working properly.
  • Gordon is now permitted to use All-Class Melees.
